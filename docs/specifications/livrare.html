Livrarea sistemului

Livrarea sistemului consta din 2 mari etape:

  1. livrarea serverului
  2. livrarea aplicatiei mobile

1. Livrarea serverului

Se va face pe computerul oferit in urmatorii pasi:

  • instalarea serverului de baze de date MySQL
  • instalarea unei versiuni de Java, preferabil Java 22
  • descarcarea arhivei WAR impachetata de dezvoltator pentru versiunea Java instalata
  • crearea unei baze de date MySQL cu numele specificat in fisierul de configurarea al aplicatiei Java ce urmeaza sa o foloseasca
  • pornirea aplicatiei intr-un terminal Java: java -jar <nume_arhiva.war>

2. Livrarea alicatiei mobile

Initial, livrarea aplicatiei mobile va avea loc doar pentru dizpozitive ce folosesc sistemul de operare Android, pe Google Play Store.
